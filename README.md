# 🚀 Galactic Invaders

> Defend Earth from endless waves of alien invaders in this fast-paced arcade shooter built with Python and Pygame.

---

## 🎮 About The Game

Galactic Invaders is a classic 2D space shooter where you pilot a starfighter against hostile alien UFOs. Destroy enemies, avoid collisions, and survive as long as possible to achieve the highest score.

---

## ✨ Features

- 👾 Endless alien invasion gameplay
- 🚀 Smooth spaceship movement
- 🔫 Laser shooting mechanics
- 🎯 Score tracking system
- 💥 Collision detection
- 🔊 Sound effects and background music
- 🕹️ Simple keyboard controls
- ☠️ Game Over screen

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/galactic-invaders.git
cd galactic-invaders
```

### 2️⃣ Install Dependencies

```bash
pip install pygame
```

### 3️⃣ Start The Game

```bash
python main.py
```
