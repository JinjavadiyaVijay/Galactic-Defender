# Galactic Invaders

A fast-paced Python & Pygame shooter where you pilot a starfighter to defend Earth from endless waves of alien UFOs. Dodge incoming fire, blast enemies, and aim for the high score!

---

##  Features

- 🎮 Classic arcade-style gameplay
- 👽 Randomized alien movements
- 💥 Laser shooting with sound effects
- 🧠 Score tracking and Game Over screen
- 🕹️ Keyboard controls: Arrow keys + Spacebar

---

## Built With

- Python 3.x
- Pygame
